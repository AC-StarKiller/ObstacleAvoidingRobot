# Description
Repository for Arduino Obstacle Avoiding Robot

# Parts
• Arduino UNO </br>
• L298N h-bridge </br>
• 2 DC motors </br>
• Rocker switch </br>
• Ultrasonic sensor HC-SR04 </br>
• Micro servo SG90 </br>
• 2 rechargeable batteries 4.2V with battery holder </br>
